# Product Decisions

## Why Signal?

Traditional job boards create information overload.
Signal introduces transparent job ranking through a
visual signal-strength metaphor that helps users
identify relevant opportunities quickly.

## Why localStorage?

The project prioritizes UX and portability.
Using localStorage allows the application to remain
fully functional without backend infrastructure.

## Why Next.js?

- SEO optimization
- Static site generation
- Performance
- Vercel deployment support

## AI Usage

AI was used as a productivity accelerator for:
- UI generation
- Component scaffolding
- Documentation
- CI/CD workflow generation
- Iterative design exploration

The product vision, feature prioritization,
architecture decisions, and UX choices were made
by the developer.
